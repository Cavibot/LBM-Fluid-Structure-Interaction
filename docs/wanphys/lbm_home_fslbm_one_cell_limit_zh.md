# HOME-FREE / FSLBM：单格尺度液面找平的固有极限

> 文档版本：2026-07-13  
> 相关实现：`wanphys/_src/fluid/fluid_grid/lbm/backends/moment/home_fp32_ref/`  
> 相关算例：`wanphys/examples/lbm/fluid_grid_lbm_dambreak_vof.py --backend home`  
> 论文映射：锐界面自由表面（HOME-FREE / Körner 型 FSLBM），路线图 G4

---

## 1. 现象（算例观察）

溃坝冲击与多轮晃荡之后，主体液池可以接近水平，但**贴壁处经常残留比内侧只高约 1 个格子的“薄沿 / 高膜边缘”**，在宏观速度已经很小（近静止）时仍长期不平齐。

常见观感：

- 不是冲到池顶的爬升柱，而是**比自由面只高一点点**的贴壁台阶；
- 接触线（高膜边界与容器壁的交线）**钉住**，液体不会沿壁外侧流回池面；
- 提高分辨率（如 64³、128³）后，同一物理缺陷被分辨得更清楚，有时还伴随溅射碎界面与质量缓慢漂移。

这不是单纯“参数没调好就能消掉”的显示问题，而是当前**单相锐界面算法在格子离散下的能力边界**。

---

## 2. 算法在做什么（以及它故意不做的事）

HOME-FREE / Körner 型 **FSLBM（free-surface LBM）** 的核心选择是：

| 做 | 不做 |
|----|------|
| 只演化液相（液体 + 一层界面格） | **不演化气相动力学** |
| 气侧用自由面重构（如 Eq.11 型）补分布 | 无真实气压、无气液双向耦合 |
| 用 φ / mass 做 VOF 式填充分数 | 无亚格子薄膜方程 |
| 可选曲率 → 拉普拉斯压修正 `ρ_g` | **无完整润湿 / 接触角壁面本构**（本仓库亦未实现） |

因此：大尺度溃坝、冲击、晃荡可以“效果不错”；但**最终静置找平**依赖的是液相内部的静水梯度与界面质量交换，而不是空气把液膜“压回去”或沿壁排走。

---

## 3. 为什么“只高一格”特别难平

### 3.1 驱动力随高度差消失

静水驱动量级大致是 `ρ g Δh`。当 `Δh` 只剩 **1 个格子** 时：

- 格子单位下 `|g|` 已为稳定性缩得很小（算例中还按柱高做了 `g ∝ 1/n`）；
- `ρ g Δh` 与数值黏性、no-slip 壁面阻力、界面层数值耗散同量级甚至更小；
- 宏观速度落到 `~10⁻³` 后，Körner 质量通量也几乎停转，接触线失去“滑动”所需的对流。

于是系统停在一个**亚稳态**：整体像平了，贴壁还高一格。

### 3.2 单相自由面在 1 格尺度上没有足够的气–液梯度

真实薄液膜回落需要：

- 气相压力 / 黏性剪切；
- 或明确的润湿势（接触角 → 弯液面形状）；
- 或亚格子薄膜模型。

FSLBM 用常数（或曲率修正）的气侧边界代替气相。当界面厚度只剩 **少数格甚至一格** 时：

- 法向“气–液”跳跃被压在一个格子宽度上；
- 填充分数 `φ` 的梯度无法再分辨“该排走的薄膜”和“合法的自由面台阶”；
- 填满 / 排空（IF/IG）阈值在 `|mass − ρ|` 很小时也不再触发。

**液相自行模拟自由面**在粗尺度有效，在**最终一格**上恰恰缺少支撑气液转换的梯度信息。

### 3.3 格子本身让这一层“显得很明显”

连续介质里 1 格对应的物理厚度可以很小；但在可视化与高度统计（按 `φ>0.5` 取柱顶）里，**一个格子就是一级台阶**。  
因此算法停在 `O(Δx)` 的残余不平，肉眼和 `hA/hB`、`h_p2p` 都会一直看见。

提高 `n` 只能把同一类缺陷挪到更细的物理尺度上，**不能 magically 消去“最后一层界面格”的离散本质**——除非换模型（两相、润湿、或亚格子薄膜）。

### 3.4 壁面与质量交换的数值钉扎

补充机制（本实现中常见）：

- **No-slip** 竖壁：接触线切向速度被抹掉，小 `Δh` 更难滑动；
- **不与墙做 Körner 交换**：墙边 `mass` 与矩得到的 `ρ` 易不一致，φ 容易钉在错误填充分数；
- **I–I 的 θ = ½(φ+φₙ)**：φ→0 时通量更弱，薄沿更难消失。

这些会加重 1 格残余，但即使全部“用力拧掉”，仍受 §3.1–3.3 的模型极限约束。

## 4. 工程上试过、且应避免的“硬找平”

为抹掉贴壁一格高沿，曾试验过例如：

- 竖壁 free-slip；
- 墙链路强制质量交换；
- 孤儿界面清理 / 贴壁膜剥离 / 凸沿原子下排。

**短期观感**有时像“膜被削掉了”，但典型代价是：

- 质量守恒被破坏（Δ 持续漂移）；
- 爬升膜与蒸发感交替出现；
- 用启发式删除界面，相当于在算法外再造一套不自洽的薄膜模型。

结论：**不应再用牺牲质量的方式强行消除 1 格残余。**  
该残余应记为 **FSLBM 验收容差**（例如高度峰峰值 `O(1)–O(2)` 格，或相对柱高的小百分比），而不是无限调参目标。

## 5. 合理预期与验收建议

| 阶段 | 合理预期 |
|------|----------|
| 冲击 / 前缘扫过 | 单相 FSLBM 可与实验/文献趋势对比 |
| 多轮晃荡 | 应看到 `hA/hB` 振荡并靠近 |
| 近静置 | 允许贴壁 **O(Δx)** 残余台阶；不必要求光学级水平面 |
| 弯液面吻壁 | 需润湿 BC；当前 HOME-FREE 路径**未承诺** |

建议日志关注：

- 冲击后质量：允许短暂波动，后期应近似平台（避免持续线性泄漏）；
- `h_p2p`、墙柱高度：收敛到小常数即可，不必逼到 0。

分辨率：

- **64³** 适合日常调试（速度与细节折中）；
- **128³** 更易看见碎界面与 1 格缺陷，需同步缩放 `|g|`（算例已 `g ∝ 1/n`）。

## 6. 若将来要真正“抹平最后一格”

按侵入性从低到高：

1. **润湿 / 接触角壁面条件**（仍在单相 FSLBM 框架内）— HOME 可选 `vof_wall_wetting`，**仅贴壁界面格、1×**。过强疏水或近角放大易把四面液面往内推，形成**四棱台**外形，算例默认 `0`；
2. **亚格子薄膜排液**（`vof_wall_film_drain`）— 对连通弯液面无效，且易泄漏 `Σmass`，默认关闭；
3. **两相 LBM / 真气相**（气可挤压与带走膜，成本高）；
4. 仅可视化层对 `φ` 做展示平滑（**不改变**守恒质量场）。

贴壁 O(Δx) 残余与墙角钉扎仍记为方法极限；不要用会造四棱台的强 κ 润湿去“抹平”。

在完成可靠润湿标定之前，把“贴壁高一点点却钉住”写成**已知限制**仍成立；上述开关是实验性减轻手段，不是保证光学级液面的承诺。

---

## 7. 一句话总结

> HOME-FREE FSLBM 用液相 + 自由面边界代替空气，大尺度自由面动力学好用；但在近静置时，**一格量级的贴壁高度差缺少足够的气–液梯度与驱动力**，格子离散又把这一层放大成可见台阶。这是方法边界，不是单靠调 `τ/γ/g` 或删界面能干净消掉的问题。
